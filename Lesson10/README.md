开发者MAC电脑里的神兵利器

[Lantern ＝ 点着灯笼翻墙](https://github.com/getlantern/lantern)

[FileZilla = 老牌Sourceforge开源文件传输软件](https://sourceforge.net/projects/filezilla/)

[SourceTree = Atlassians出品的图形化版本管理工具，支持Git和Mercurial](https://www.sourcetreeapp.com/)

[JD-GUI = Java的class文件反编译神器，可以从二进制class文件查看它的Java源代码](http://jd.benow.ca/)

[JADX = JD-GUI的增强版，支持查看安卓apk/dex文件中反编译的Java源代码以及查看apk中其他文件的内容](https://github.com/skylot/jadx)

[Charles = 网络抓包利器加上代理功能，并支持自签名证书，所以可以用来在手机上抓取https的包。使用非常方便。付费软件](https://www.charlesproxy.com/)

[Wireshark = 老牌网络抓包利器，各种平台都可以玩耍](https://www.wireshark.org/)

[Sublime Text = 轻量级的编辑器](http://www.sublimetext.com/)
